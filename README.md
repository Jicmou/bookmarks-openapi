# Bookmarks-openapi

## Installation

```bash
git clone https://github.com/Jicmou/bookmarks-openapi.git <YOUR_DIRECTORY>
cd <YOUR_DIRECTORY>
npm install
```

## Start Locally
```bash
npm start
```
